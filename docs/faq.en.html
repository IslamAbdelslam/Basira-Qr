Frequently Asked Questions

Why do I need a VirusTotal API key?

To allow URL analysis via VirusTotal for free within usage limits.

Why is the APK size big?

Universal APKs include all ABIs. Use AAB to reduce download size from the store.

What happens offline?

Nothing is sent to VirusTotal; the URL is passed directly with a friendly notice.

What permissions are required?

Camera, Internet, and Network State only.

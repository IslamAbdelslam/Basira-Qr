Quick Start

  1. Install the app.
  2. Enter your VirusTotal API key in Settings.
  3. Scan a QR code; the URL is analyzed and a result is shown.

Settings

HTTPS Warning

Warns for HTTP (non-secure) links.

URL Scanning

When enabled, checks URLs via VirusTotal before opening.

Offline Mode

If no internet, nothing is sent to VirusTotal and the URL is passed directly.

البدء السريع

  1. قم بتثبيت التطبيق.
  2. أدخل مفتاح VirusTotal API في الإعدادات.
  3. امسح رمز QR؛ سيتم تحليل الرابط وإظهار النتيجة.

الإعدادات

تحذير HTTPS

تحذير عند الروابط التي تستخدم HTTP غير الآمن.

مسح الروابط

عند تفعيله، يتم فحص الروابط عبر VirusTotal قبل الفتح.

وضع عدم الاتصال

عند عدم توفر الإنترنت، لن يتم الاتصال بـ VirusTotal وسيتم تمرير الرابط مباشرة.

## 🛡️ نظرة عامة

بصيرة كيو آر هو ماسح رموز QR آمن وحديث يتكامل مع منصة ذكاء التهديدات في VirusTotal لتحليل الروابط قبل فتحها. مبني باستخدام React Native و Expo، يوفر تجربة مستخدم سلسة مع ميزات أمان شاملة، ودعم ثنائي اللغة، وإمكانيات العمل في وضع عدم الاتصال.

## ✨ الميزات الرئيسية

### 🔒 الأمان والخصوصية
- **تحليل الروابط في الوقت الفعلي**: التكامل مع VirusTotal API v3 للكشف الشامل عن التهديدات
- **تحذيرات HTTPS**: تنبيهات تلقائية للاتصالات HTTP غير الآمنة
- **التخزين الآمن**: مفاتيح API والبيانات الحساسة مخزنة بأمان باستخدام Expo SecureStore
- **الخصوصية أولاً**: لا توجد تتبع، جميع البيانات معالجة محلياً أو عبر APIs آمنة

### 🌍 التدويل
- **الدعم ثنائي اللغة**: ترجمة كاملة للعربية والإنجليزية مع دعم تخطيط RTL
- **تبديل اللغة الديناميكي**: تغيير اللغة دون إعادة تشغيل التطبيق
- **التكيف الثقافي**: عرض نص مناسب وتعديلات واجهة المستخدم للغات المختلفة

### ⚡ الأداء وتجربة المستخدم
- **حجم حزمة محسن**: APK أصغر بنسبة 40-60% من خلال تقنيات التحسين المتقدمة
- **وضع عدم الاتصال الذكي**: معالجة أنيقة عند عدم توفر اتصال الإنترنت
- **دعم الثيمات**: مخططات ألوان متعددة مع خيارات الوضع المظلم/الفاتح
- **سجل المسح**: تتبع وإدارة آخر 20 رابط تم مسحها

### 🎯 الوظائف الأساسية
- **مسح عالي الأداء**: تكامل كاميرا محسن للكشف السريع عن رموز QR
- **التحقق من الروابط**: كشف وتنظيف وتحقق ذكي من الروابط
- **الوعي بالشبكة**: كشف الاتصال التلقائي مع سلوك احتياطي ذكي
- **إعدادات قابلة للتخصيص**: تبديل ميزات المسح وتخصيص تفضيلات المستخدم

## 🚀 البدء السريع

### المتطلبات المسبقة
- Node.js 18+ و npm
- Expo CLI أو EAS CLI
- Android Studio (لتطوير Android)
- مفتاح VirusTotal API مجاني

### التثبيت
```bash
# استنساخ المستودع
git clone https://github.com/IslamAbdelslam/BasiraQr.git
cd BasiraQr

# تثبيت التبعيات
npm install

# بدء خادم التطوير
npm start
```

### التطوير
```bash
# بدء خادم تطوير Expo
npm start

# تشغيل على Android
npm run android

# تشغيل على iOS (يتطلب macOS)
npm run ios
```

## 🔧 التكوين

### إعداد VirusTotal API
1. احصل على مفتاح API مجاني من [VirusTotal](https://www.virustotal.com/gui/join-us)
2. شغل التطبيق
3. انتقل إلى **الإعدادات → تكامل VirusTotal → تغيير مفتاح API**
4. أدخل مفتاح API الخاص بك (يتم التحقق منه تلقائياً قبل الحفظ)

## 🏗️ البنية المعمارية ومجموعة التقنيات

### التقنيات الأساسية
- **الإطار**: React Native 0.81.4 مع Expo SDK 54.0.12
- **اللغة**: JavaScript (ES6+) مع React 19.1.0
- **إدارة الحالة**: React Context API مع useReducer
- **التنقل**: React Navigation v6
- **التخزين**: Expo SecureStore + AsyncStorage

### الأمان والشبكات
- **ذكاء التهديدات**: تكامل VirusTotal API v3
- **مراقبة الشبكة**: React Native NetInfo لكشف الاتصال
- **عميل HTTP**: Axios للاتصالات الآمنة مع API
- **التحقق من البيانات**: التحقق من الروابط وتنظيفها المخصص

### تحسينات الأداء
- **تقسيم الكود**: Metro bundler مع tree shaking
- **تحسين الأصول**: صور وموارد مضغوطة
- **تحسين البناء**: ProGuard، تقليص الموارد، وبناءات خاصة بالمعمارية
- **إدارة الذاكرة**: دورة حياة مكون محسنة وإدارة حالة

## 📱 دعم المنصات

### Android
- **الحد الأدنى للإصدار**: Android 5.0 (مستوى API 21)
- **الإصدار المستهدف**: Android 14 (مستوى API 34)
- **المعمارية**: ARM64، ARMv7، x86_64
- **الأذونات**: الكاميرا، الإنترنت، حالة الشبكة

### iOS (مخطط)
- **الحد الأدنى للإصدار**: iOS 13.0
- **الإصدار المستهدف**: iOS 17.0
- **الحالة**: قيد التطوير

## 🔒 اعتبارات الأمان

### حماية البيانات
- مفاتيح API مخزنة بأمان باستخدام keychain/keystore الجهاز
- لا يتم نقل بيانات حساسة دون تشفير
- معالجة محلية للعمليات غير الحساسة
- جمع واحتفاظ بالحد الأدنى من البيانات

### أمان الشبكة
- اتصالات API HTTPS فقط
- ربط الشهادات للنقاط الحرجة
- التحقق من الطلبات/الاستجابات
- آليات انتهاء الصلاحية وإعادة المحاولة

## 📊 مقاييس الأداء

- **حجم APK**: ~80-95MB (محسن من 170-200MB)
- **سرعة المسح**: 0.5-1 ثانية في المتوسط

- **تأثير البطارية**: استخدام خلفي ضئيل
- **وقت البدء**: <1.5 ثانية بدء بارد
